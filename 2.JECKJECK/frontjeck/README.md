# frontjeck

## 프로젝트 생성
- bootstrapVue를 쓴다면 버전 2로 생성해야함.
- 아래의 명령어 대로 설치 할 것 (router버전 포함)
```
--vue 버전 2설치
npm install vue@2
npm install vue@2.6.14

--router

npm install vue-router@3.5.3
npm install vuex@3.6.2

--프로젝트 생성
npm install -g @vue/cli
vue create 프로젝트명

```
